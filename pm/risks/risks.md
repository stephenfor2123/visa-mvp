## 每周评审 checklist

每周五 standup 末尾 30min,过一遍 12 条:

- [ ] 概率 / 影响 是否变?变 → 重排紧急度
- [ ] 缓释动作是否完成?完成 → 标 `mitigated`
- [ ] 是否已经触发过?触发过 + 已处理 → 标 `closed`
- [ ] 是否有新风险?有 → 加 R13, R14...
- [ ] Owner 写好下周缓释动作(给下周 standup 看)

## 状态流转

```
open  ──(缓释动作完成,且未触发)──>  mitigated
open  ──(触发 + 已有 plan)───────>  in_progress
in_progress  ──(已解决)──────────>  closed
mitigated  ──(3 周内未触发)───────>  closed
open/mitigated  ──(Owner 失联)──>  escalated(由 PM 接管)
```
