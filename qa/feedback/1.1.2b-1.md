### 2. ❌ API 调用契约不匹配 (P0 — 阻断)

- **症状**: MaterialsValidate.vue 永远显示错误态,主 UI 不可达
- **现状**:
  - `MaterialsValidate.vue:251-256` 调 `validateMaterials({material_ids:[], applicant_data:{}})` (对象)
  - `materials.js:170-173` 函数签名 `validateMaterials(materialIds)` 期望数组
  - `Array.isArray(obj) === false` → 必抛 'materialIds must be a non-empty array'
- **修复**: 二选一
  - (A) caller 传数组 `validateMaterials([1,2,3])` + 后端再 wrap
  - (B) callee 改签名 `validateMaterials({material_ids, applicant_data})` 接受对象
- **推荐**: 选 (A) 更简洁,callee 自己 fetch material_ids 后再做

### 3. ❌ mock 数据没 error 严重度 (P1 — 演示)

- **症状**: mock 只能产生 1 warning + (N-1) pass,severity 集合 = {warning, pass}

  - 重新拍摄按钮 (119 行) → dead code
