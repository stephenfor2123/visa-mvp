⚠️ 3 个隐藏风险(法务/合规必看)

1. 3D Secure 2.0(PSD2 / SCA):欧盟 + 英国 + 印度强制要求,无验证拒付率从 1.2% 升至 0.6%。Visa / Master 必须启用,App 走 Stripe SDK 即可。
2. PCI DSS Level 1:支付数据 永远不能触碰明文卡号。前端用 Stripe Elements / Braintree,后端只收 token。违规罚款 $5K-$100K/月。
3. 汇率损失:跨境收款通常损失 0.8-1.5pp。优先选支持多币种钱包的平台(PingPong / Airwallex),锁定 T+0 中间价。年节省 $5K-$50K 视规模。

📚 常见问题

拒签了能全额退款吗?
我们提供"按效果付费"服务:被拒签后,App 自动退还 30% 服务费,同时触发平安拒签险理赔(¥200/¥500 定额)。退款会在 7 个工作日内原路返回至您的支付账户。具体见 《拒签险条款》。
签证申请材料能保证真实性吗?
所有酒店/机票单据均通过 Booking / Skyscanner Affiliate 真实渠道出具,含 PNR + 取消号,可在第三方网站独立验证。App 绝不提供任何伪造 / PS / OCR 改写功能——此类行为在申根 / 美 / 英 / 日 / 澳均构成移民欺诈,最高 10 年监禁 + 永久拒签。
DS-160 表格怎么填?需要多久?
DS-160 是美国 B1/B2 签证的官方申请表,共 9 大模块 50+ 字段。系统已通过护照 OCR + 预填模板自动填好 82%,你只需检查并补全剩余项,平均填写时间 4-10 分钟。
个人信息是否安全?会泄露吗?
所有数据使用 AES-256 + KMS 加密存储于 AWS Singapore(主)+ Mumbai(备),7 年保留期。仅签证官 + 你本人可访问。第三方 DPO(OneTrust / TrustArc)持续监控。我们同时遵守 GDPR(欧洲) + PIPL(中国) + 多国本地法规。
App 支持哪些支付方式?
中国版:银联 / 微信支付 / 支付宝 / Visa-Master。Visa / Master 卡通道走 Stripe,自动启用 3D Secure 2.0 强验证。海外版:PayPal / Razorpay + UPI(印度)/ Doku(印尼)/ Visa-Master。
我能在哪些国家使用 App?
目前支持 9 国主推:美国、申根 27 国、日本、英国、澳大利亚、加拿大、韩国、新西兰、中国(部分)。35+ 目的地全覆盖。MVP 阶段首发印度 + 印尼 + 越南。

最后更新:2026-06-01 · 版本 v2.3 · 适用于所有地区用户

1. 引言

Atlys Visa Services(以下简称"我们")高度重视用户的隐私权。本隐私政策说明我们如何收集、使用、存储、共享和保护您的个人信息,以及您所享有的权利。

本政策同时遵守:欧盟《通用数据保护条例》(GDPR)、中国《个人信息保护法》(PIPL)、美国《加州消费者隐私法》(CCPA)、印度《数字个人信息保护法》(DPDP) 以及目的地国家本地法规。

2. 我们收集的信息

2.1 你主动提供

  • 账号信息:手机号、邮箱、姓名(中英)、身份证件号
  • 签证申请:护照、签证申请表、照片、行程、家庭信息、工作信息
  • 支付信息:通过 Stripe / PayPal / Razorpay 等第三方平台,我们不直接存储你的银行卡号
  • 客服沟通:在线聊天、邮件、电话录音(征得同意后)

2.2 自动收集

  • 设备信息(型号、操作系统、唯一设备标识符)
  • 使用日志(IP 地址、访问时间、页面浏览、点击行为)
  • Cookies 与同类技术(详见 Cookie 政策)

3. 数据使用目的

  • 提供签证申请服务(包括 OCR 识别、模板填写、单据生成、订单跟踪)
  • 处理支付与退款
  • 申请拒签险理赔
  • 客服支持与争议解决
  • 合规审计与反欺诈
  • 产品改进(经脱敏与聚合处理)
  • 法律要求的其他用途

4. 数据存储与跨境传输

主存储:AWS Singapore(新加坡)。备份:AWS Mumbai(印度)。

对于欧洲用户,跨境传输依据 EU 2021 标准合同条款(SCC) 与 转移影响评估(TIA) 进行。

对于中国大陆用户,数据 不出境,独立部署于合规区域。

保留期:账号活跃期间 + 注销后 30 天软删除 + 7 年备份保留(签证材料合规要求)。

5. 第三方共享

  • 政府机构:仅在你授权签证申请时,将必要信息提供给目的地国家签证中心(VFS / TLS / BLS / CGI 等)
  • 保险公司:仅在你购买拒签险后,将必要信息提供给平安 / Allianz
  • 支付机构:Stripe / PayPal / Razorpay / Doku 等
  • 云服务商:AWS
  • 客服外包:经 DPA 协议约束的本地化客服

我们不出售你的个人信息,从不与广告商共享。

7. 安全措施

  • 传输加密:TLS 1.3

拒签险服务条款

保单号示例:PA-20260610-2847 · 平安 · 亚洲 200 元定额

1. 产品说明

拒签险是第三方保险产品,由平安保险(中国版)或 Allianz(海外版)承保。Atlys 作为保险中介,代用户采购、绑定保险,并在出险时代为提交理赔。

App 自身不是保险人,不承担保险责任。

2. 产品规格(平安版)

  • 亚洲 200 元定额:保费 40 元,赔付 200 元
  • 全球 500 元定额:保费 70 元,赔付 500 元
  • 保险期:≤ 60 天
  • 生效时间:出单后立即生效,覆盖整个签证申请期

3. 投保条件

  • 年龄 18-70 周岁
  • 持有有效中华人民共和国居民身份证(中国版)
  • 身体健康,能正常出行
  • 未被任何国家列入拒签黑名单

4. 理赔条件

满足以下全部条件方可理赔:

  • 通过 Atlys App 完成签证申请并支付服务费
  • 保险产品在保单有效期内
  • 提供拒签信原件(签证官签字 + 大使馆盖章)
  • 未通过任何其他途径(包括其他保险公司)获得拒签赔付
  • 不属于"除外责任"中的任何一条

5. 理赔流程

  1. 系统自动检测到拒签(从签证中心 API + 邮件确认)
  2. 系统代为提交理赔申请(7 日内)
  3. 保险公司审核(7-15 工作日)
  4. 保险公司打款到用户预留账户(¥200 或 ¥500)

总耗时:平均 14-21 个工作日。

6. 除外的责任

以下情况不予理赔:

  • 因申请人提供虚假材料导致拒签
  • 因申请人犯罪记录导致拒签
  • 因申请人健康问题(如传染病)被拒绝入境
  • 因申请人目的地国家政策突变(如新增签证类型限制)导致拒签
  • 因不可抗力(战争、动乱、自然灾害)导致拒签
  • 申请人主动撤回申请
  • 保单到期后才发生的拒签
  • 申请人被列入任何国家黑名单后投保

7. 隐私保护

投保信息(姓名、身份证号、护照号)仅在理赔必要时提供给平安 / Allianz。不用于其他营销用途。

理赔争议应首先联系保险公司客服(平安 95511 / Allianz 4008-888-888)。如协商不成,可向中国银保监会投诉或提交仲裁。
