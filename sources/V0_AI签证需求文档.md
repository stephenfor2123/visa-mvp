AI校验通过后，加载对应国家官方表单模板； 2. 已识别字段**自动填充至表单对应位置**，保留手动编辑入口； 3. 用户确认表单无误，点击「提交申请」； 4. 前端触发RPA调度，自动访问目标签证官网； 5. 自动处理：图形验证码、滑动验证码（**无活体检测**）； 6. 提交中展示进度动画，提交成功/失败分弹窗提示。 ### 3.6 订单状态查询模块 1. 提交完成后，生成唯一订单编号； 2. 系统定时拉取签证官网状态，前端实时展示： 待提交 → 已提交 → 审核中 → 出签 → 拒签 3. 状态变更主动消息推送（App/小程序通知）； 4. 超时订单前端标记「状态异常」，仅展示提醒，不自动处理。 ### 3.7 增值跳转模块（非核心） 1. 订单流程内增加入口：拒签险、国际机票、境外酒店； 2. 仅做**外链跳转**，不做深度接口对接； 3. 跳转页面在新窗口打开，不脱离本系统主流程。 --- ## 第四章 业务中台（后端服务模块 详细需求） ### 4.1 用户服务 1. 账号管理：注册、登录、注销、信息修改、密码重置； 2. 权限管控：区分普通C端用户，无B端权限； 3. 数据关联：账号与订单、材料一对一绑定； 4. 注销逻辑：触发定时任务，72小时清空该账号所有数据&文件。 ### 4.2 订单服务 1. 订单生命周期管理：创建、提交、状态同步、完结、异常标记； 2. 订单编号：全局唯一编码，规则后台固定； 3. 状态定时拉取：配置轮询周期（可后台修改），从签证官网同步状态； 4. 异常订单统计：后台汇总超时、提交失败订单。 ### 4.3 文件服务 1. 文件接收、转码、缩略图生成； 2. 本地对象存储归档，按订单/用户分目录存储； 3. 加密处理：所有上传文件自动加密，禁止明文存储； 4. 自动清理任务：临时文件24h删除、归档文件180h删除； 5. 文件复用支持：同一用户历史材料可重复调取使用。 ### 4.4 配置服务（后台核心配置项） 1. 国家配置：新增/下架国家、绑定对应签证类型、表单模板； 2.
