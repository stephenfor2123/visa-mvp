# Design References

UI 设计参考图存档，方便后续查找和复用。

## 命名规范

`<section>-<style>-v<n>.png`

- `section` — 页面/区块名（如 `destinations-hero`）
- `style` — 风格描述（如 `vibrant` / `minimal` / `dark`）
- `v<n>` — 版本号
